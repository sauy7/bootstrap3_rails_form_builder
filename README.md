= Bootstrap3RailsFormBuilder

## Requirements
* Ruby 1.9+
* Rails 4.0+
* Twitter Bootstrap 3.0+

## Installation

Add it to your Gemfile:

`gem 'bootstrap3_rails_form_builder' github: 'sauy7/bootstrap3_rails_form_builder'`

Run the following command to install it:

`bundle`
